# TWThreat

Turtle WoW 仇恨监控插件，实时显示队伍/团队中每个人对当前目标的仇恨值。

使用条件：处于**小队**或**团队**中，攻击**精英**怪物或 **Boss**。

## 安装

1. 下载并解压
2. 将 `TWThreat` 文件夹复制到 `游戏目录\Interface\AddOns\`
3. 重启游戏（或在角色选择界面点击"插件"确认已启用）

> **注意**：GitHub 下载的 zip 解压后文件夹名可能是 `TWThreat-main`，需要重命名为 `TWThreat`，否则游戏无法识别。

## 基本使用

进入小队或团队后，选中一个精英怪/Boss 并进入战斗，仇恨窗口会自动显示每个人的仇恨排名。

### 窗口操作

| 操作 | 说明 |
|------|------|
| **拖拽标题栏** | 移动窗口位置 |
| **拖拽右下角** | 调整窗口大小 |
| **滚轮滚动** | 当仇恨列表超出窗口时上下滚动 |

### 标题栏按钮

| 按钮 | 功能 |
|------|------|
| 齿轮 | 打开设置面板 |
| 锁 | 锁定/解锁窗口位置 |
| X | 关闭窗口 |

## 功能特性

### 仇恨显示
- 实时显示每个队友的仇恨值、TPS（每秒仇恨）、仇恨百分比
- 条形颜色按职业区分，你自己的条始终为红色便于识别
- 顶部的 "Pull Agro At" 条显示你还需要多少仇恨才会 OT

### 目标发光
在默认 UI 或 pfUI 的目标头像上显示渐变光环，直观反映仇恨状态：
- 绿色 → 黄色：0% - 49%（安全）
- 黄色 → 红色：50% - 100%（危险）

### 仇恨百分比
在目标头像上直接显示你的仇恨百分比数值。

### 全屏发光警告
仇恨达到 OT 阈值时，屏幕边缘会闪烁红色光晕提醒你注意仇恨。支持自适应屏幕分辨率。坦克模式下自动禁用。

### OT 警告声音
仇恨达到 OT 阈值时播放警告音效。坦克模式下自动禁用。

### 坦克模式
显示一个额外窗口，列出你正在拉住的所有怪物，以及每个怪物仇恨列表中下一个玩家的名字和百分比，方便坦克快速判断是否需要换目标嘲讽。

> 开启坦克模式后，全屏发光和 OT 警告声音会自动禁用。

### 战斗显示 / 战斗隐藏
可设置战斗开始时自动显示窗口，战斗结束后自动隐藏，保持界面整洁。

## 设置面板

点击窗口标题栏的齿轮按钮打开设置。打开设置时会自动显示预览数据，方便你实时查看调整效果。

### 外观

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 条形材质 | 状态条的视觉样式（4 种可选） | 2 |
| 条形间距 | 条形之间的间距（0-10） | 0 |
| 行高 | 每个条形的像素高度（10-30） | 15 |
| 柔和配色 | 使用低饱和度的柔和色调 | 关 |
| 色彩饱和度 | 职业颜色的鲜艳程度（1-20） | 10 |
| 显示背景 | 窗口是否显示背景 | 开 |

### 警告

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 全屏发光 | 仇恨过高时屏幕边缘红色闪烁 | 关 |
| 警告声音 | 仇恨过高时播放提示音 | 关 |
| OT 阈值 | 触发警告的仇恨百分比（65-95%） | 85% |

### 显示

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 标签行 | 显示列标题（TPS、仇恨、百分比） | 关 |
| TPS 列 / 仇恨列 / 百分比列 | 单独控制各列的显示/隐藏 | 开 |
| 战斗透明度 | 战斗中窗口的透明度（50-100%） | 100% |
| 非战斗透明度 | 脱战后窗口的透明度（50-100%） | 100% |
| 窗口缩放 | 整体缩放比例（50-150%） | 100% |

## 聊天命令

以下命令均以 `/twt` 开头：

```
/twt show             显示主窗口
/twt toggle           切换窗口显示/隐藏
/twt lock             切换窗口锁定
/twt reset            重置仇恨数据
/twt texture [1-4]    设置条形材质
/twt spacing [0-10]   设置条形间距
/twt scale [50-150]   设置窗口缩放百分比
/twt saturation [1-20] 设置色彩饱和度
/twt pastel           切换柔和配色
/twt backdrop         切换窗口背景
/twt tankmode         切换坦克模式
/twt who              查询团队中谁安装了此插件
/twt debug            切换调试模式
```

也可以用 `/twtshow` 直接显示主窗口。
